# Быстрый старт

## 1. Установка зависимостей

```bash
npm install
```

## 2. Запуск в режиме разработки

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## 3. Использование

1. Нажмите кнопку "Войти" на главной странице
2. Введите логин и пароль от аккаунта Anixart
3. После авторизации вы увидите экраны с вашими итогами года
4. Используйте кнопки "Назад" и "Далее" для навигации
5. На последнем экране нажмите "Скачать PNG" для экспорта

## 4. Настройки

- Нажмите на иконку настроек (⚙️) в правом верхнем углу для изменения темы
- Выберите цветовую схему (автоматическая/светлая/тёмная)
- Выберите цветовую палитру
- Настройки сохраняются автоматически

## 5. Сборка для продакшена

```bash
npm run build
npm start
```

## Примечания

- Токен авторизации хранится в sessionStorage (удаляется при закрытии браузера)
- Настройки темы сохраняются в localStorage
- Экспорт PNG работает только для текущего экрана
- Размер экспортируемого изображения: 1080×1920 px
